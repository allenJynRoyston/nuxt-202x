## Install instructions:
```sh
$ git clone https://github.com/allenRoyston/nuxt-202x.git
$ cd nuxt-202x
$ npm install 
$ npm run dev
$ // open browser at http://localhost:1234
```

## Unit testing
Make sure Jest is installed globally before running unit tests.
``` sh
$ npm install jest -g
$ npm run test
```
